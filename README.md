# Hello World Interactive App

A simple interactive web application built with HTML, CSS, and JavaScript.

## Features

- **Click Counter**: Track your clicks with an animated counter
- **Background Color Changer**: Cycle through beautiful gradient backgrounds
- **Name Greeting**: Get a personalized greeting message

## How to Use

1. Clone this repository
2. Open `index.html` in your browser
3. Interact with the features on the page

## Technologies Used

- HTML5
- CSS3 (with animations and gradients)
- Vanilla JavaScript

## Live Demo

Open `index.html` in any modern web browser to see the app in action.
